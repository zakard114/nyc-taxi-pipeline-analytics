# `data/github/` — GitHub Archive sample (local)

Generated by this repo’s GitHub track:

1. `python dlt/github_archive_ingestion.py` → creates **`github_test.duckdb`** here (via dlt; working directory is set in that script).
2. `python scripts/export_duckdb_to_json.py` → **`github_events_100.json`**.
3. `python scripts/upload_to_gcp.py` → uploads NDJSON to GCS and loads BigQuery (needs GCP env + credentials).

`*.duckdb` is gitignored. NDJSON may be regenerated anytime.
