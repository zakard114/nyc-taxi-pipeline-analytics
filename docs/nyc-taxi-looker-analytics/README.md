# NYC Taxi — Looker Studio dashboard exports

This folder holds **static exports** of the **NYC Taxi Data Pipeline Analytics (2019–2020)** Looker Studio report.

| File | Description |
|------|-------------|
| `NYC_Taxi_Data_Pipeline_Analytics_(2019–2020).pdf` | PDF export of the dashboard |
| `NYC_Taxi_Data_Pipeline_Analytics_(2019–2020).png` | PNG image (screenshot or image export) |

The live report is backed by BigQuery marts (e.g. `dm_citywide_monthly`, `dm_service_type_totals`) with date range and optional `service_type` filters aligned to the 2019–2020 ingest scope.
